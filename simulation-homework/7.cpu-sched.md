# 4: For what types of workloads does SJF deliver the same turnaround times as FIFO?

- If any job has greater length than the one arrived before it

```
l[i+1] > l[i] where i represents arrival order
```

# 5: For what types of workloads and quantum lengths does SJF deliver the same response times as RR?

- If quantum length greater than or equal to 2nd max length of jobs _(job with max length runs last so it doesn't have to share time)_

```
q = second_max(l[i])
```

# 6: What happens to response time with SJF as job lengths increase?

- Response times of later jobs increase, since they are scheduled only after previous jobs are done.

# 7: What happens to response time with RR as quantum lengths increase? Can you write an equation that gives the worst-case response time, given N jobs?

- Response times of later jobs increase, since they are scheduled only after each previous job has run for `quantum_length` time.
- Worst case is when all previous jobs have length >= `quantum_length`.

```
. i starts from 0
. Number of jobs before i-th job: (i - 1) - 0 + 1 = i
. Worst response time of i-th job:
WRT(i) = quantum_length * num_previous_jobs
       = quantum_length * i
```
